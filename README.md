# PI - Dashboard
A dashboard for a raspberry pi server
## Features
- [x] CPU/GPU stats
- [x] Network stats
- [x] Storage stats
- [x] Stack overview
- [x] File browsing
- [x] Theme switching
- [ ] Stack control
- [ ] Settings tab
- [ ] Quite a lot..
## Usage
1. `git clone https://github.com/warreverheijen/pi-dashboard/new/master?filename=README.md`
2. `npm start`
